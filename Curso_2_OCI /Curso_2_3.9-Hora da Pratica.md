# 🧪🚀 Atividades com Postman

## 🟢 Atividade 1
- 📂 Abra o Postman e crie uma nova solicitação  
- 🌐 Insira a URL de sua escolha  
- 🔽 Selecione o método **GET**  
- 📤 Clique em **Send**  
- 🔍 Analise a resposta:
  - 📦 Body (corpo)
  - 🧾 Headers (cabeçalhos)
  - 📊 Status Code

## 🔵 Atividade 3
- 📂 Crie uma nova solicitação  
- 🌐 Insira a URL  
- 🔽 Escolha o método **POST**  
- 📝 No Body:
- Selecione **raw**
- Escolha formato **JSON**
- 📦 Insira dados JSON válidos:
```
json
{
  "exemplo": "valor"
}
```
📤 Clique em Send
🔍 Analise como os dados são enviados

## 🟣 Atividade 4
🔁 Utilize uma requisição anterior (GET ou POST)
📊 Após executar, vá em Headers
🔍 Analise:
📄 Content-Type
